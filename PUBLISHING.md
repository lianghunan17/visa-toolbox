# 发布说明

这是一套可公开发布的静态网页工具。

## 发布定位

适合作为：
- 个人工具站
- 小型签证辅助工具页
- 演示版原型
- 社群内部分享页

## 不适合作为

- 法律意见服务
- 官方承诺型查询系统
- 带真实敏感用户数据的生产系统

## 发布前建议检查

- 页面免责声明是否保留
- 是否仍含本地测试地址
- 是否仍含不希望公开的私人案例数据
- JSON 数据是否都是可公开内容

## 推荐发布文案

可在发布说明里写：

> 本工具基于日本出入国在留管理庁公开资料与整理规则制作，仅供参考，不构成法律意见或官方承诺。请以最新官方信息及实际审查结果为准。

## 推荐上线方式

优先使用 Cloudflare Pages。

发布后建议测试：
- 首页是否可打开
- `/visa` 是否正常
- `/pr` 是否正常
- JSON 是否正常加载
- PDF 导出是否可用
- 手机端页面是否正常显示
