# 签证工具箱最小后台说明

## 当前已完成

后台页面：
- `admin.html`
- `admin.js`

当前支持：
- 保存管理员 token
- 查看授权列表
- 搜索授权
- 手动创建授权
- 停用授权
- 重置设备绑定
- 查看评价统计
- 查看评价明细

---

## 本地打开方式

先启动前端静态站：

```bash
cd pr-app
python3 -m http.server 8000
```

然后访问：

```txt
http://127.0.0.1:8000/admin.local.html
```

说明：
- `admin.local.html` 只用于本地联调，已自动注入本地 API 地址
- `admin.html` 作为正式部署入口，不再默认写死本地 API

---

## 使用前提

后端 `license-api` 必须已启动，并且：
- `ADMIN_BEARER_TOKEN` 已配置
- Supabase 表已建好

---

## 使用步骤

### 1. 输入管理员 token
这个 token 要和后端 `.env` 里的：

```txt
ADMIN_BEARER_TOKEN=
```

保持一致。

### 2. 点击保存 token
保存后页面会自动拉取授权列表。

### 3. 新建授权
填：
- 用户邮箱
- 用户姓名（可选）
- 订单号（可选）
- 商品名

点击创建后，会直接返回激活码。

### 4. 管理授权
列表中可直接：
- 停用授权
- 重置设备绑定

### 5. 查看评价
后台中可直接看：
- 评价统计
- 评价明细

如果后端接口可用，会优先显示真实汇总评价。
如果后端暂时不可用，会回退显示当前浏览器本地评价。

---

## 当前定位

这是最小后台，不追求完整权限系统。

当前目标只是：
- 让你能管理激活码
- 让你能查订单对应授权
- 让你能在用户换设备时快速处理
